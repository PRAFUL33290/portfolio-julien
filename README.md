# Portfolio de Julien GUERRIER

Ce projet est un portfolio personnel de Julien GUERRIER, un Creative Technologist et Content Manager basé à Parempuyre. Le site présente ses compétences, ses projets et ses informations personnelles.

## Structure du projet

Le projet contient les fichiers suivants :

- **index.html** : Ce fichier contient la structure HTML du portfolio. Il inclut le lien vers le CDN de FontAwesome, le header avec les informations personnelles, la section des compétences avec des barres de progression, la section des projets et le footer.
  
- **css/style.css** : Ce fichier contient le style CSS pour le portfolio. Il définit les styles pour le fond, les couleurs d'accent, le design responsive, et les barres de progression pour les compétences.
  
- **assets/images** : Ce dossier est destiné à contenir les images utilisées dans le portfolio, comme les icônes ou les logos.

## Déploiement sur GitHub Pages

Pour déployer ce portfolio sur GitHub Pages, suivez ces étapes :

1. Créez un nouveau dépôt sur GitHub.
2. Clonez le dépôt sur votre machine locale.
3. Copiez tous les fichiers du projet dans le répertoire cloné.
4. Commitez vos changements :
   ```bash
   git add .
   git commit -m "Ajout du portfolio"
   ```
5. Poussez les changements vers GitHub :
   ```bash
   git push origin main
   ```
6. Allez dans les paramètres de votre dépôt sur GitHub.
7. Sous la section "Pages", sélectionnez la branche `main` et le dossier `/ (root)`.
8. Cliquez sur "Save".

Votre portfolio sera accessible à l'adresse `https://<votre-nom-utilisateur>.github.io/<nom-du-dépôt>/`.
